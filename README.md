## üìñ Project Overview

This project is a comprehensive data visualization dashboard built using **Tableau** to analyze Airbnb data. The goal of this analysis is to provide actionable insights into pricing trends, customer preferences, and property performance across different regions. 

By leveraging historical hosting data, this dashboard helps real estate investors and hosts understand the factors that drive higher ratings and pricing power.

## üéØ Problem Statement

The short-term rental market is highly competitive. Hosts and investors need data-driven answers to questions such as:
- *Which neighborhoods command the highest prices?*
- *How does the number of bedrooms correlate with price?*
- *What is the trend of host registrations over the years?*
- *Which regions have the best review scores?*

This project tackles these questions by transforming raw data into an interactive visual story.

## üìä Key Features & Insights

This Tableau dashboard provides a deep dive into the following key performance indicators (KPIs) and metrics:

### 1. **Geospatial Pricing Analysis**
   - **Visualization**: Map-based visualization of average prices by Zipcode/Region.
   - **Insight**: Identifies premium locations where property values and rental rates are highest, assisting in location scouting for new investments.

### 2. **Price vs. Room Type**
   - **Visualization**: Bar charts comparing average prices across 'Entire home/apt', 'Private room', and 'Shared room'.
   - **Insight**: Highlights the revenue potential of different property types, showing a clear premium for entire home rentals.

### 3. **Temporal Trends (Year-over-Year)**
   - **Visualization**: Line graphs tracking the average price changes over time (Year of Host Since).
   - **Insight**: reveals market inflation trends and the growth of the Airbnb ecosystem over the years.

### 4. **Review & Rating Distribution**
   - **Visualization**: Analysis of 'Top Regions based on User Rating'.
   - **Insight**: Correlates location with guest satisfaction, identifying areas where hosts consistently perform well.

### 5. **Availability & Capacity**
   - **Visualization**: 'Top Regions based on Bed‚Äôs availability'.
   - **Insight**: Assesses supply saturation in various neighborhoods.
